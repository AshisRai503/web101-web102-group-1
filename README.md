# Task Manager — WEB101/WEB102 Group Project

A full-stack task management web app built with the PERN stack.

## What I Built
- Login and Signup pages with form validation
- Dashboard with stats, donut chart, bar chart, and calendar
- Manage Tasks page with status filtering
- Create Task page with form
- Team Members page
- Calendar page
- Sidebar navigation with active route highlighting

## Tools and Technologies Used
- **Next.js** (App Router) - Web application framework
- **Tailwind CSS** - Styling
- **React Hook Form** - Form handling and validation
- **Chart.js + react-chartjs-2** - Dashboard charts
- **React Icons** - Sidebar icons

## How to Run the Project

### Step 1: Clone the repository
```bash
git clone https://github.com/your-username/your-repo.git
```

### Step 2: Go to the project folder
```bash
cd task-manager
```

### Step 3: Install dependencies
```bash
npm install
```

### Step 4: Run the development server
```bash
npm run dev
```

### Step 5: Open in browser
```
http://localhost:3000
```
